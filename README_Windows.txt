Timer Assistant - Windows Installation Guide
============================================

QUICK START:
-----------
1. Double-click "TimerAssistant.bat" to run the app
2. That's it! The timer app will open automatically.

REQUIREMENTS:
------------
- Windows 10 or 11
- Python (will be installed automatically if needed)

HOW TO USE THE TIMER:
--------------------
Type commands in natural language:

Examples:
- "set a 5 minute timer for coffee break"
- "start a 25 min pomodoro timer" 
- "create 1 hour meeting timer"
- "pause the coffee timer"
- "show all timers"
- "stop meeting timer"

TROUBLESHOOTING:
---------------
If the app doesn't start:
1. Make sure you have internet connection (for Python installation)
2. Run as administrator if needed
3. Install Python manually from https://python.org/downloads/
   (Make sure to check "Add Python to PATH" during installation)

FEATURES:
--------
- Natural language commands
- Multiple timers running at once
- Visual timer display in right panel
- Sound alerts when timers complete
- Pause/resume functionality
